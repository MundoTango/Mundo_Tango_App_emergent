## Why Were Agents Missed Initially?

### Root Causes:
1. **Focused only on infrastructure**: Initial search targeted server-side agents (ESA layers)
2. **Missed client-side agents**: Page agents live in frontend routes
3. **Overlooked specialized agents**: Journey agents, algorithm agents, UI sub-agents are in different files
4. **Didn't search "The Pages" documentation**: Key docs folder wasn't explored
5. **No comprehensive grep**: Needed to search for P*, J*, A*, MB*, ESA* patterns

### Learning for MB.MD:
- **Always search multiple patterns**: Don't just search "agent" - search P\d+, J\d+, A\d+, MB\d+
- **Check all documentation folders**: docs/, The Pages/, archived/
- **Read route configurations**: Routes often map to page agents
- **Search for "125", "journey", "algorithm"**: These keywords indicate specialized agents
- **Use parallel grep searches**: Search multiple patterns simultaneously
